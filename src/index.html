Interactive pandemic simulation to encourage critical thinking

Explore the spread of diseases through an interactive simulation tool that allows you to model different strategies and observe their effects.

About This Project

The project is designed to promote critical thinking about pandemics and the spread of diseases rather than predict real world outcomes. This tool allows regular people such as journalists to create simplified epidemiological models that users can easily access and use them to experiment with different scenarios and assumptions.

Get Started

Click the button below to launch the configuration generator. You can then create your own pandemic simulation by customizing population parameters, disease parameters, intervention strategies, and scenarios.

Examples

Feel free to explore these pre-configured simulations. Each example demonstrates a different use-case for which the simulation can be used.

  • COVID-19 Challenge - Test your skills by balancing public health with economic considerations.
  • Comparison of Proposals - Compare different strategies side-by-side.
  • Foot-and-Mouth Disease Simulation - Experiment with the versality of the simulation framework by simulating FMD.
